This is an Ember addon that wraps the date picker [flatpickr](http://chmln.github.io/flatpickr/).

Compatibility
------------------------------------------------------------------------------

* Ember.js v3.20 or above
* Ember CLI v3.20 or above
* Node.js v12 or above

Installation
------------------------------------------------------------------------------

```
ember install ember-flatpickr
```
